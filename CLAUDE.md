# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Repository Purpose

This is a GitHub profile repository (the special `username/username` repo displayed on the user's GitHub profile). It contains the user's public profile introduction, contact information, and skills.

## Files

- **README.md** — The main GitHub profile content. Uses HTML/Markdown to format the introduction, social links, and technical skills.
- **portfolio_banner.png** — A banner image displayed at the top of the profile.

## Common Tasks

### Updating Profile Content
- Edit README.md directly to update bio, skills, social links, or any profile information.
- Use HTML and Markdown mixed (this is standard for GitHub profile READMEs).
- Keep external image URLs pointing to reliable sources (GitHub raw content, CDNs); avoid broken image links.

### Changing the Banner
- Replace portfolio_banner.png with a new image file of the same name, or update the image source path in README.md.

## Important Guidelines

- **Keep profile information accurate and up-to-date** — This is a public-facing profile seen by potential employers, collaborators, and the open-source community.
- **Test links before committing** — External links in the README should be verified to work (social links, profile images, etc.).
- **Profile consistency** — If updating social media handles, ensure they're consistent across the profile and actual social accounts.
